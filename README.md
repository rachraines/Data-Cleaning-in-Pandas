# Data-Cleaning-in-Pandas
A Python data cleaning project using Pandas to standardize, clean, and prepare customer data for analysis. Includes handling missing values, formatting phone numbers, splitting address fields, and exporting a cleaned dataset.


This project is a complete data cleaning workflow built in Python using **Pandas**.  
The dataset (customer information) was cleaned, standardized, and prepared for future analysis.
